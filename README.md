# Installation

Install the required packages:
````
poetry install
````

# Usage

Run streamlit.
````
poetry run streamlit run babyagi.py 
````

You can now view your Streamlit app in your browser.

Local URL: http://localhost:8501

To stop the Streamlit server, press ctrl-C.
